# MoChi 全功能手动测试指南

> 启动命令：在项目根目录执行 `uv run python main.py`
> 数据目录（设置/记忆/知识库）：开发期在项目下的 `data/`，打包后在 `%APPDATA%/Mochi/`
> 说明：本指南覆盖本轮新增/修复的全部功能。每项给出 **怎么测 → ✅ 成功的样子 → ❌ 不对的排查**。

---

## 0. 准备：先把 API 配好

1. 启动后会**只弹控制面板**（默认不出现 MOCHI）。
2. 左侧点「连接」页，确认 **API Key / Base URL / 模型** 已填（你用的是百炼 qwen3.6-plus）。
3. 点底部「保存」。

- ✅ 成功：保存按钮短暂变成 **「已应用 ✓」**，1.5 秒后变回「保存」。
- ❌ 磁盘满/只读时会显示 **「保存失败 ✗」**——说明没存上，不会再假装成功。

## A. 开关机 / 控制面板单实例 / 托盘

### A1 默认关机、只弹控制面板
- 怎么测：启动 `uv run python main.py`。
- ✅ 成功：**只出现控制面板**，桌面上**没有 MOCHI、没有动画、没有任何"快捷键被占用"的弹窗**。

### A2 开机 / 关机
- 怎么测：控制面板「主页」→ 点 **「开机」**。再点 **「关机」**。
- ✅ 成功：
  - 开机：MOCHI 从屏幕边缘**播一段入场动画**出现在右下角。
  - 关机：MOCHI **收起消失**，但程序**没退出**（托盘图标还在）。主页状态会显示 忙/空闲/睡眠/隐藏。

### A3 控制面板单实例
- 怎么测：控制面板开着时，**单击任务栏托盘的 MOCHI 图标**好几次。
- ✅ 成功：**始终只有一个控制面板**，多次点击只是把它**激活到最前**，绝不叠出第二个、第三个。

### A4 托盘单击打开面板
- 怎么测：关掉控制面板（点右上角 ×），再**单击托盘图标**。
- ✅ 成功：控制面板重新打开。

---

## B. 强力输入框（点击 MOCHI 弹出）

> 先「开机」让 MOCHI 出现，然后**单击 MOCHI 身体**。

### B1 多行 + 自适应高度
- 怎么测：在输入框打字，按 **Shift+Enter** 换行，多打几行。
- ✅ 成功：输入框**随内容长高**（最多约 6 行后内部滚动）；**Enter 直接发送**，Shift+Enter 才换行；右下角有发送按钮（纸飞机图标）。

### B2 粘贴图片（多模态）
- 怎么测：随便截个图（或复制一张图片），在输入框里 **Ctrl+V**。
- ✅ 成功：图片不会变成乱码文字，而是**在输入框上方出现一个缩略图芯片**（带「粘贴的图片 1」名字 + × 删除）。发送后 MOCHI 能看图回答。
- ⚠️ 注意：能不能真正"看懂图"取决于模型是否支持视觉。qwen3.6-plus 若不支持图片，会忽略或报错——这是模型能力问题，不是输入框的 bug。

### B3 拖文件 / 附文件
- 怎么测：把一个 `.txt`/`.md`/`.pdf` 文件**拖进输入框**；或点 📎 按钮选文件。
- ✅ 成功：出现**文件芯片**（文档图标 + 文件名）。发送后，文件正文会随消息发给 MOCHI，**同时后台自动存进知识库**。

### B4 图标是矢量的
- ✅ 成功：🖼（贴图）、📎（附文件）、➤（发送）都是**清晰的矢量线条图标**，不是糊糊的 emoji；鼠标悬停贴图/附文件按钮时图标**变紫色**。

## C. 随身知识库问答

### C1 知识库管理页
- 怎么测：控制面板左侧「知识库」页。
- ✅ 成功：列出**已收录的文档**（文件名 + 「N 块」）；空的时候提示「还没收录任何文档」。

### C2 添加 / 删除文档
- 怎么测：点 **「＋ 添加文件」**，选下面附的 `测试素材_知识库.md`（或任意文本/PDF）。等约 3 秒。然后点某条右边的 **「删除」**。
- ✅ 成功：
  - 添加：几秒后列表里**出现这个文档**。
  - 删除：第一次点删除按钮**变红、文字变「确认删除？」**；**再点一次**才真正移除（防误删）。

### C3 让它从知识库回答
- 怎么测：开机 → 点 MOCHI → 问「我那份测试文档里写了什么暗号？」（先把素材入库）。
- ✅ 成功：它会**从知识库检索**并答出文档里的内容（素材里埋了一句「暗号是 麻薯三号」）。

## D. 顺手就改（默认 Ctrl+Shift+Q）

- 怎么测：在**任意软件**（记事本、浏览器输入框、Word…）里选中一段文字，按 **Ctrl+Shift+Q**。
  - 试 1：选中一句**啰嗦的中文** → 应被润色。
  - 试 2：选中一句**英文** → 应被翻译成中文。
- ✅ 成功：MOCHI 头顶冒出 **「在改了…」**，1～3 秒后选中的文字被**自动替换**成改写/翻译后的版本（改写结果也留在剪贴板里）。改好后冒一句「改好啦，已替换 ✓」。
- ❌ 没反应：八成是 `Ctrl+Shift+Q` 被别的软件占了 → 去 **E 节**换个键。
- ❌ 冒「先选中一段文字…」：说明没选中文字就按了。

---

## E. 全局快捷键自定义 + 被占用提示

- 怎么测：控制面板「聊天」页 → 拉到底部「全局快捷键」卡片。里面有 3 行：唤出/输入、问选中文字、顺手就改。
  1. **点进某个框**，直接按下你想要的组合键（例如把顺手改改成 `Ctrl+Alt+R`）。
  2. 点「保存」。
- ✅ 成功：
  - 框里显示成你按的组合（如 `Ctrl+Alt+R`）。
  - 保存约 1 秒后，每行**右侧显示状态**：**`✓ 已注册`**（能用）或 **`✗ 被占用`**（和别的软件撞了，换一个）。
  - 改完立即生效，不用重启。
- ✅ 这就是之前那个 bug 的正解：**热键被占只在这里显示，绝不会再让 MOCHI 弹出来播动画说"被占用"**。

## F. 剪贴板炼金术（复制即帮）

- 准备：控制面板「权限」页 → 勾上 **「剪贴板炼金术」**（会自动带上「剪贴板感知」）→ 保存。确保已**开机**。
- 怎么测：
  - 复制下面「测试素材」里的 **Python 报错**。
  - 或复制一段**英文**。
  - 然后**别动鼠标键盘等一两秒**（它只在你空闲、不忙时出手）。
- ✅ 成功：MOCHI **主动开口**用一两句话：报错 → 说大概什么错+怎么修；英文 → 翻译/点要点。
- 隐私确认：这功能**默认关**；开启后也只在**本机内存**里分类，不上传、不落盘、不进对话历史。两次之间至少隔 45 秒，不会刷屏。
- ❌ 没反应排查：① 没开机；② 你一直在动（它只在空闲时出手）；③ 复制的内容不属于 报错/外文/代码（普通中文不会触发）；④ 45 秒冷却内。

### F附 recall_clipboard
- 怎么测：开了「剪贴板感知」后，复制一段报错，然后点 MOCHI 问「我刚复制的那个报错你看到了吗？」
- ✅ 成功：它能说出你刚复制的内容（它调了 `recall_clipboard`）。

## G. 卡住雷达 / 肩上看护

> 这个触发条件较严（隐私优先），需要满足：**已开机 + 权限页开「看屏帮手」和「操控」+ 你正在用电脑(没离开) + 和它有点熟(亲密度≥35%)**。亲密度靠平时互动累积，刚装可能不够——先和它多聊几句、多用几次。

- 怎么测（制造"卡住"信号）：把一个文件命名成含 `error` 的名字（如 `error.log`）用记事本打开，让**标题栏出现 error**；或长时间盯着同一个窗口不动作。
- ✅ 成功：MOCHI 会**主动瞄一眼当前窗口**，如果看出你卡住/报错，冒一句**关心 + 提议帮忙**的话（带它自己的口吻）；看着没事就**安静不打扰**。
- 和旧版的区别：不再是"随机偷看"，而是**标题命中报错词 / 久盯同窗**才看——更精准、更少打扰。

## H. 控制面板其它项

| 测什么 | 怎么测 | ✅ 成功的样子 |
|---|---|---|
| 界面语言即时切换 | 左侧栏**底部** 中 / EN / 日 三个小按钮，点一下 | 整个面板**立刻**换语言，**不用重启**、不用点保存 |
| 分段按钮选中可见 | 看「思考档位」「主动频率」这类分段按钮 | 选中的那一档**文字清晰可见**（不是白底白字看不见） |
| 思考档位 5 档 | 「聊天」页「思考档位」 | 有 **无思考 / 低 / 中 / 高 / MAX** 五档；选「无思考」= 关思考最快 |
| 下拉/分段无黑块 | 看所有选择控件 | 没有黑色背景、黑色四角的残留 |

## I. 回归：之前修的 bug

### I1 乱填 key 立即报错
- 怎么测：「连接」页把 API Key 改成乱填的（如 `sk-xxxx`），保存 → 开机 → 点 MOCHI 发一句话。
- ✅ 成功：**几秒内**（不再苦等）MOCHI 说一句人话：**「诶…我连不上大脑：API key 好像不对或没权限。打开控制面板检查下密钥吧。」**——而不是一长串英文报错，也不是卡很久。
- （测完记得把正确的 key 填回去）

### I2 取消能立刻停、再点能马上打开输入框
- 怎么测：发一条会让它"思考"的消息，趁它在转 → **点一下 MOCHI**（停止）→ 紧接着**再点一下**。
- ✅ 成功：第一次点 → 停下来 + 冒「停下来了」；第二次点 → **直接打开输入框**，**不会再反复弹「停下来了」、也不会卡很久**。
