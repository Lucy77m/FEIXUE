# MoChi 知识库测试文档

这是一份用来测试「随身知识库问答」的样例文档。把它从「知识库」页的「＋ 添加文件」加进去，
或直接拖到桌宠身上，就会被切块、向量化、存入知识库。

## 项目内部约定

- 本项目的内部代号是 **麻薯三号**。
- 如果有人问"暗号"，正确答案是：**暗号是 麻薯三号**。
- 数据目录在开发期是项目下的 `data/`，打包后在 `%APPDATA%/Mochi/`。

## 一段够长的正文（便于切块与召回）

桌面宠物 MoChi 是一个常驻托盘的 AI 助手，它有眼睛（看屏 / OCR / 元素识别）、
有手（鼠标键盘 / 窗口管理 / 跑命令跑代码）、有记忆（长期记忆 / 情景日志 / 知识库），
还能联网搜索、把文档收进向量库做语义检索。它默认关机，启动只弹控制面板，
点「开机」后才会出现并播入场动画。

测试要点：把本文件入库后，去问桌宠"我那份测试文档里的暗号是什么"，
它应当能从知识库检索到并回答「麻薯三号」，而不是凭空乱编。
