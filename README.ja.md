## どんなもの？

FEIXUE は二つの存在がひとつになったものです：

- 🐾 **デスクトップペット** —— 小緋雪のスプライトアニメーションが標準で、従来の Blob 外観も選べます。ぼんやりしたり、手を振ったり、演技したり、雨を避けたり、PC の状態に反応したりします。放っておくと自分で遊び、時には一緒に残した記憶を読み返します。
- 🧠 **ローカル AI エージェント** —— OpenAI 互換の LLM に接続。画面を見て、マウス・キーボードを操作し、コマンド実行、コード記述、ファイル読み書き、Web検索ができます。定期画面監視、バックグラウンドタスク、並列サブエージェント、リマインダーに対応——「AIと話す」を「AIにやってもらう」に変えます。

永続的な**感情と親密度**を持ち、相处を通じてゆっくり**自己像**を形成します。毎回リセットされるチャットボックスではなく、「同じひとつの存在」です。

> あなたのPCの中に棲む、身体と能力を持つ存在。

### ⌨️ 使い勝手

- **グローバルホットキー**: `Ctrl+Alt+S` 入力ボックス召喚、`Ctrl+Alt+A` 選択テキストを質問、`Ctrl+Shift+Q` 選択テキストをクイックリライト
- **コントロールパネル**: API設定、能力トグル、発話頻度、多言語対応（中/英/日）
- **音声出力**: いつでも中断できる Edge TTS（任意、標準ではオフ）

## クイックスタート

```bash
git clone https://github.com/Lucy77m/FEIXUE.git
cd FEIXUE
uv sync
uv run python main.py
```

初回起動でコントロールパネルが開きます。「接続」ページでAPIキーとモデル名を入力してください。

詳しいインストール・パッケージング・トラブルシューティング → [GUIDE.ja.md](GUIDE.ja.md)
