# MoChi 全功能清单（当前版本）

> 一句话：MoChi 是一个常驻 Windows 托盘的**桌面宠物 AI Agent**——有眼睛（看屏/OCR/元素识别）、有手（鼠标键盘/窗口/跑命令跑代码）、有记忆（长期记忆/情景日志/向量知识库）、能联网，外加一套生产力杀手锏与可爱的桌宠交互。默认关机，启动只弹控制面板，点「开机」才出现。

---

## 一、Agent 核心：自然语言 → 多步工具链

一句话下指令，Agent 自动规划并多步调用工具完成（最多 16 步，可调，硬上限 64）。实时可见思考过程（思绪粒子）、计划清单（黑板）。

**全部 54 个工具，按用途分组：**

- **执行/系统**：`run_shell`、`run_python`、`install_package`、`system_memory`、`read_process_memory`
- **文件/代码**：`read_file`、`write_file`、`edit_file`、`list_dir`、`search_code`、`glob_files`
- **联网**：`web_search`、`web_fetch`、`http_request`
- **屏幕感知**：`screenshot`、`ocr_screen`（带坐标）、`find_on_screen`（图标定位）、`screen_elements`+`act_element`（最强 GUI 自动化：标号点击）
- **鼠标键盘**：`click`、`double_click`、`right_click`、`move_mouse`、`scroll`、`type_text`（中文自动走剪贴板）、`press_keys`
- **窗口**：`list_windows`、`focus_window`、`manage_window`（最小/最大/移动/关闭）
- **剪贴板**：`read_clipboard`、`write_clipboard`、`recall_clipboard`（读"最近一条有趣剪贴板"）
- **记忆**：`remember`、`recall`、`set_preference`、`note_env`、`forget_memory`（自纠错）
- **知识库**：`ingest_docs`、`recall_docs`、`list_docs`、`forget_docs`
- **提醒/任务**：`schedule_reminder`（到点亲口提醒）、`schedule_task`（到点自动执行）
- **并发**：`spawn_agent`（子代理，≤4 并发）、`start_background_task`（后台任务，≤3 并发）
- **表演/展示**：`perform`（做动作/剧情）、`show_image`、`play_gif`、`plan`（计划清单）、`confirm`（危险操作前请你批准）
- **MCP**：`mcp__<服务>__<工具>`（用户在 mcp.json 配的外部连接器，条件性出现）

## 二、⭐ 生产力四杀手锏（本轮新增）

| 功能 | 触发 | 作用 |
|---|---|---|
| **顺手就改** | 选中任意软件的文字 → `Ctrl+Shift+Q` | LLM 智能改写（中文润色 / 外语翻译 / 啰嗦精简）→ 写回剪贴板并自动粘贴替换。独立调用，不碰主对话历史 |
| **剪贴板炼金术** | 复制到报错/外文/代码（默认关，权限页开） | 空闲时桌宠主动用一两句解释/翻译/说明。本地分类、内存内、不上传、45 秒节流 |
| **随身知识库问答** | 控制面板「知识库」页 / 拖文件给桌宠 | 列出/添加/两段式删除已收录文档；问它时从向量库语义检索 |
| **卡住雷达 / 肩上看护** | 标题命中报错词 / 久盯同窗（需开看屏+操控+开机+亲密度≥35%） | 证据驱动地瞄一眼当前窗口，看出卡住就主动提议帮忙；没事就安静 |

## 三、强力输入框（点击桌宠弹出）

- **多行 + 自适应高度**：Enter 发送 / Shift+Enter 换行 / Esc 关闭
- **粘贴/拖拽图片**（多模态）：Ctrl+V 或拖入 → 缩略图芯片；超大图自动转 JPEG 控体积
- **附带文件**：拖文件 / 📎 选文件 → 正文随消息发出，并后台入知识库
- **矢量图标**：贴图/附文件/发送都是 SVG 图标（hover 变紫），非 emoji
- 附件去重、粘贴图自动编号

## 四、记忆 · 学习 · 人格（关系驱动）

- **长期记忆**：经验/教训/偏好/环境事实，对话时自动召回（向量检索）
- **情景日志**（journal）：记住你俩近来一起经历的事，聊天自然带出
- **自我画像演化**（persona）：相处中慢慢长成"自己"，每轮反思缓慢更新
- **后台反思**：每轮对话后独立线程汲取经验、更新记忆/偏好/画像（不阻塞聊天）
- **情绪引擎**：效价/唤醒/亲密度三维；识别表扬/批评→情绪动作；随时间衰减；亲密度需互动维持

## 五、桌宠交互与动画

- **动态形象**（QPainter 实时绘制）：眨眼/看人、6 种表情、20+ 动作、**7 个迷你剧情**（coffee/fish/sleuth/read/music/game/stars）
- **入场动画**、**睡眠/清醒**（久不动会睡）、**自娱自乐**（无人时做梦/做小剧情）
- **拖拽 + 贴边躲藏**（拖到屏幕边自动收起，并收起跟随的浮层）
- **气泡系统**：打字机语音气泡、思考进度气泡、思绪粒子、黑板（Markdown 结构化展示）、拍立得相框（图/GIF）
- **确认面板**：危险操作"执行/不执行"二选一

## 六、主动行为 · 节日

- **主动搭话**（频率可调：安静/正常/话痨）：6 种模式 `check_in / follow_up / share_day / thought / late_care(深夜关心) / welcome_back(欢迎回来)`，会结合你当前窗口选切入点；主动时有概率真去网上瞄一眼再分享
- **节日 + 生日**：元旦/情人节/愚人节/儿童节/万圣节/平安夜/圣诞节/跨年夜 + 你的生日（生日格式容错，"6-2"也认）→ 庆祝 + 应景的话

---

## 七、提醒 · 后台 · 子代理

- **提醒**（到点亲口说，含 2 小时迟到宽限）、**定时任务**（到点真执行）
- **后台任务**（长活儿丢后台、不卡聊天，做完播报，≤3 并发）
- **子代理**（独立子任务派工作线程，≤4 并发，不可再派）
- **技能库**：把验证过的脚本存成技能复用（create/run/edit/list_skills）

## 八、语音

- **TTS**（Windows SAPI，无新依赖）：回复/提醒/主动消息可朗读，默认关，自动选中文嗓子

## 九、全局热键（✨ 可在控制面板自定义）

| 默认键 | 功能 |
|---|---|
| `Ctrl+Alt+S` | 唤出 MOCHI + 输入框（关机时顺便开机） |
| `Ctrl+Alt+A` | 选中文字 → 快速问它 |
| `Ctrl+Shift+Q` | 顺手就改 |

- 在「聊天」页底部用按键捕获框**自定义任意组合键**，保存即生效。
- 每个键右侧显示 **`✓ 已注册` / `✗ 被占用`**（被占用只在面板提示，绝不再用桌宠动画打扰）。

## 十、控制面板（浅色侧边栏导航）

**页签**：主页（开关机/状态/心情/亲密度）· 它眼中的你（画像/偏好/经验/统计）· **知识库（管理文档）** · 连接（API/模型/思考档位）· 聊天（语言/温度/步数/主动/TTS/**快捷键**）· 权限 · 关于（含重置）
**界面语言**：侧边栏底部 中/EN/日 **即时切换**（无需重启）

**全部 27 个设置项**：
- 连接：`api_key` `base_url` `model` `embed_model` `proxy` `temperature` `max_tokens` `max_steps` `enable_thinking` `think_level`（无/低/中/高/MAX 五档）
- 交互：`language` `ui_language` `proactive_enabled` `proactive_level` `tts_enabled` `birthday`
- 权限：`allow_web` `allow_control` `allow_shell` `watch_screen` `clip_sampler` `clip_alchemy` `clip_alchemy_kinds`
- 热键：`hotkey_summon` `hotkey_ask` `hotkey_quick`
- 其它：`quick_paste_back`

## 十一、安全 · 隐私 · 权限

- **三大权限开关**：联网 / 操控（鼠标键盘屏幕进程）/ 命令执行（shell/python/技能/写文件），可分别关闭
- **确认前置**：删文件、覆盖重要文件、`git push --force`、系统改动等必须先 `confirm`
- **看屏帮手/剪贴板感知默认关**，且有亲密度门槛、冷却、内存内不落盘
- **截图自动排除 MOCHI 自己的窗口**
- **原子写入**：所有持久化用临时文件+替换，半截崩溃不留坏文件
